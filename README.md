# glcParaFNC

Este repositório é dedicado ao trabalho prático da disciplina de Linguagens Formais e Autômatos.

## Funcionamento

- Leitura do arquivo
  - Parsing das entradas
- Transformações em FNC
  - Remoção do símbolo inicial
  - Remoção do lambda
  - Remoção de regras de cadeia
  - REACH
  - TERM
  - FNC
